# Project
React 19 SPA with Vite, Zustand, React Router, Tailwind, i18next.

# Commands
- Dev: pnpm dev
- Tests: pnpm test:coverage
- Build: pnpm build
- Preview: pnpm preview

# Code Style
- 2-space indentation
- Named exports
- Components: src/components/
- Pages: src/pages/
- Services: src/services/
- Hooks: src/hooks/
- Store: src/store/
- Utils: src/utils/
- Tests: src/test/

# Key Technologies
- State: Zustand
- Routing: React Router v7
- i18n: i18next + react-i18next
- Styling: Tailwind CSS v4
- Testing: Vitest + @testing-library
- PWA: vite-plugin-pwa + workbox-window
